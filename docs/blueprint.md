# **App Name**: Barcode Stocktake

## Core Features:

- Add Products: Add items to the inventory via manual entry or by scanning a barcode/QR code, with fields for product name, quantity, and barcode data.
- Inventory List: Present inventory in sortable list with product details. Allows search and filtering.
- Remove Products: Allows removing products from the stocktake.
- A detailed dashboard with a stock overview: A detailed dashboard with a stock overview
- Can prepair invoice when a product is sold: Can prepair invoice when a product is sold

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) to evoke trust and stability.
- Background color: Light blue-gray (#F0F4F7) for a clean and modern feel.
- Accent color: Analogous blue (#34A853) to add emphasis without clashing.
- Body and headline font: 'Inter', sans-serif.
- Use clear, simple icons for actions like 'add,' 'remove,' and 'scan,' following the Material Design style.
- Employ a clean, card-based layout for product display to enhance readability and scannability.
- Incorporate subtle transitions on interactions, like adding or removing items, to provide visual feedback.
